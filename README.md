# AZ Nail Tech Website

A polished, responsive static website for a trendy Arizona nail tech.

## Pages included

- `index.html` — homepage
- `services.html` — nail service overview
- `pricing.html` — service menu and pricing guide
- `gallery.html` — portfolio-style design gallery
- `booking.html` — appointment request page with client-side validation
- `about.html` — nail tech story and studio standards
- `contact.html` — contact and location details
- `nail-care.html` — SEO-friendly aftercare guide
- `nail-art-trends.html` — SEO-friendly trend page

## Files included

- `styles.css` — full responsive design system
- `script.js` — mobile navigation, form validation, gallery filters, and small UI helpers
- `assets/logo.svg` — editable SVG logo
- `assets/favicon.svg` — browser favicon
- `robots.txt` and `sitemap.xml` — basic SEO support

## How to preview locally

Open `index.html` in a browser.

For the cleanest local preview, use VS Code Live Server or run:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Customization notes

Replace placeholder contact details, booking links, service prices, and social links before publishing. The booking form is front-end only and currently creates a friendly confirmation message; connect it to Formspree, Netlify Forms, Webflow, or a scheduling tool before using it for real bookings.

## Design direction

The website uses a soft editorial nail studio look: cream backgrounds, blush accents, chrome-inspired details, clean typography, rounded cards, high contrast text, and responsive spacing. It avoids heavy generic gradients and keeps the design specific to beauty, nails, and appointment conversion.
